question: gc（Go コンパイラ）によるコンパイル方法を教えてください。
answer: Go言語のコンパイルには、基本的に`go build`コマンドが使用されます。`go build`コマンドを使うことで、Goコンパイラ(gc)がソースコードをコンパイルし、実行ファイルを生成します。

以下、Go言語のコンパイル手順です。

1. Go言語をインストール
まず、Go言語の公式サイト（https://golang.org/dl/ ）からインストールパッケージをダウンロードし、インストールします。

2. ソースコードの作成
任意のテキストエディタを使ってGo言語のソースコードを記述します。例えば、`main.go`ファイルに以下のサンプルコードを書きます。

```go
package main

import "fmt"

func main() {
    fmt.Println("Hello, world!")
}
```

3. ターミナルまたはコマンドプロンプトを開く
ターミナル（MacおよびLinux）またはコマンドプロンプト（Windows）を開き、ソースコードが置かれているディレクトリに移動します。例：

```
cd /path/to/your/sourcecode
```

4. コンパイル
以下のコマンドをターミナルまたはコマンドプロンプトに入力して、ソースコードをコンパイルします。

```
go build
```

このコマンドは、カレントディレクトリにあるソースコードをコンパイルし、実行ファイルを生成します。実行ファイルの名前は、ソースコードのディレクトリ名になります。

5. 実行ファイルの実行
コンパイルされた実行ファイルを使ってプログラムを実行します。例：

- MacおよびLinuxの場合:

```
./your_executable_name
```

- Windowsの場合:

```
your_executable_name.exe
```

これで、Go言語のソースコードがコンパイルされ、実行されます。この場合、画面に`Hello, world!`と表示されるでしょう。
